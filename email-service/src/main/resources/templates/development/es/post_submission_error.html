Hubo un problema al enviar tu declaración de impuestos federales del 2024 al IRS

Hubo un problema técnico cuando enviaste tu declaración con Direct File y no pudimos enviar tu declaración al IRS. Guardamos tu información, pero tendrás que encontrar otra forma de enviar tu declaración de impuestos federales.

      Iniciar sesión en Direct File      

La fecha límite para enviar la declaración de impuestos federales es el 15 de abril a las 11:59 PM en tu zona horaria (o el 17 de abril a las 11:59 PM para los residentes de Massachusetts). Si te preocupa poder cumplir con la fecha límite para enviar la declaración, podrías solicitar una prórroga o extensión de tiempo para declarar tus impuestos.

Si ya pasó la fecha límite para la declaración de impuestos federales y no programaste un pago, envía el pago lo antes posible para limitar las multas.
