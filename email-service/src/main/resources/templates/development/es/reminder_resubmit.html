Direct File

Reenvía tu declaración del impuesto federal rechazada

Enviaste una declaración del impuesto federal con Direct File, pero el IRS rechazó tu declaración.

Si ya la reenviaste o encontraste otra forma de enviarla y el IRS la aceptó, no necesitas hacer nada más.

La fecha límite para reenviar tu declaración con Direct File es el 20 de abril del 2024 a las 11.59 PM. Después del 20 de abril, necesitarás otra forma de enviar tu declaración.

Cómo reenviar

Inicia tu sesión en Direct File para revisar tu declaración rechazada y conocer qué hacer ahora. Si tienes preguntas o necesitas asistencia, chatea con un representante de asistencia al cliente de Direct File desde tu tablero.

      Inicia tu sesión en Direct File      

¿Qué pasa si necesito más tiempo?

Puedes aplicar a una extensión hasta el 15 de abril (o hasta el 17 de abril para residentes de Masachusets). No podrás usar Direct File para enviar una extensión.

No puedes usar Direct File si tienes una extensión o planeas enviar tu declaración después de la fecha límite federal para enviar declaraciones. Necesitarás encontrar otra forma de enviar tu declaración del impuesto federal. Recuerda que aún si tienes una extensión para enviar tu declaración, necesitas pagar los impuestos que debes antes de la fecha límite para evitar penalidades.
