Don't forget to file your 2024 state taxes

If you haven't already filed state taxes, remember to file by April 15, 2024 (or April 17 for Massachusetts residents).

How to file your state taxes

Arizona

Visit Arizona's FileYourStateTaxes website to transfer your federal tax return data and file your state taxes by April 15, 2024.

California

Visit California's CalFile website to file your state taxes by April 15, 2024.

Massachusetts

Visit Massachusetts' MassTaxConnect website to transfer your federal tax return data and file your state taxes by April 17, 2024.

New York

Visit New York's FileYourStateTaxes website to transfer your federal tax return data and file your state taxes by April 15, 2024.
