What if I need more time?

You can apply for an extension by April 15. You won't be able to use Direct File to file for an extension.

Remember, even if you have an extension to file your tax return, you still need to pay your taxes by the original due date to avoid interest and penalties for failing to pay on time.

Don't forget about state taxes and credits

Depending on your situation, you may have to file a state tax return in addition to a federal tax return. You may also be eligible for state tax credits.

If you need to file a state return, sign in to Direct File to be guided to your state's free tax filing tool.

Keep due dates for state tax returns in mind. Some state tax filing tools may not allow you to electronically file your state tax return until your federal tax return has been accepted.

Note that if you've already filed your state tax return, you may need to file an amended state return if you entered incorrect data from your rejected federal return.
